# Shelf Aura Book Shop

## Project Overview
This is a static website for a book shop called "Shelf Aura". It showcases various book categories and featured books with a modern, responsive design.

## Files Analysis

### Book_Shop.html
- **Structure**: Contains navigation, hero section, categories, most read books, trending books, and footer.
- **Features**: 
  - Responsive navigation bar with links to sections.
  - Hero section with welcome message and call-to-action button.
  - Categories section with 6 book genres (Fiction, Self Help, Romance, Science, Biography, Comics).
  - Two book sections: "Most Read" and "Trending Now", each displaying 3 books with cover images.
  - Footer with copyright information.
- **Technologies**: HTML5, links to external CSS and Google Fonts.

### style.css
- **Design**: Modern, clean design with gradient backgrounds and subtle shadows.
- **Key Styles**:
  - Uses Poppins font family.
  - Color scheme: Purple accents (#6a5acd) on white and gradient backgrounds.
  - Responsive grid layouts for categories and books.
  - Hover effects: Navigation links change color, category cards lift and change color, book cards scale up.
  - Sticky navigation bar.
  - Hero section with overlay background image.
- **Layout**: Flexbox and CSS Grid for responsive design.

## Technologies Used
- HTML5
- CSS3 (with modern features like CSS Grid, Flexbox, and transitions)
- Google Fonts (Poppins)

This project demonstrates basic web development skills with a focus on responsive design and user experience.
